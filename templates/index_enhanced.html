üìö System Overview

This Military AI Screening System uses deep learning to analyze sensor data from wearable devices and assess candidate suitability for military roles.

üéØ Features

  • Single Candidate Screening: Test individual candidates with demo data
  • Batch CSV Processing: Upload CSV files to screen multiple candidates at once
  • Biomarker Extraction: Movement quality, fatigue index, and smoothness analysis
  • Role Recommendations: AI-driven military role suggestions based on performance
  • Risk Assessment: Identify potential health and injury risks

üìÑ CSV File Format

Your CSV file should contain:

  • Column 1: candidate_id (optional but recommended)
  • Columns 2-562: 561 sensor features from accelerometer and gyroscope data

Download the template for the correct format!

üî¨ How It Works

  1. Data Collection: Sensor data from wearable devices during physical activities
  2. Preprocessing: Data is normalized and scaled
  3. AI Analysis: 1D-CNN model analyzes movement patterns
  4. Biomarker Extraction: Key health and fitness indicators are extracted
  5. Decision Making: Knowledge graph maps biomarkers to military roles

üìä Decision Criteria

  • PASS (>80% confidence): Excellent physical performance, suitable for combat roles
  • CONDITIONAL PASS (60-80%): Adequate performance, suitable for support roles
  • FAIL (<60%): Poor performance, requires medical evaluation
